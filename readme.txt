Organize WordPress media library files into nested folders with filtering, bulk moves, import/export, and role-aware controls.

== Description ==

MY Files PRO adds folder organization to the WordPress Media Library using a private hierarchical taxonomy. It includes a folder sidebar, Media Library filtering, folder management actions, drag/drop assignment, bulk moves, import/export tools, and configurable role access.

== Installation ==

1. Upload the `my-files-pro` folder to `/wp-content/plugins/`.
2. Activate MY Files PRO from the Plugins screen.
3. Go to Settings > MY Files PRO to configure access, counts, default upload folder, import/export, and uninstall cleanup.
